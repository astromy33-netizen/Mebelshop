## Формат URL:
```
https://images.unsplash.com/photo-{ID}?w=800&h=600&fit=crop&q=80
```

## Как добавить свои фотографии:

1. Найдите изображение на Unsplash.com
2. Скопируйте ID фотографии из URL
3. Замените в файле `src/data/sampleProducts.js` в поле `cover`

Или используйте другие бесплатные источники:
- Pexels.com
- Pixabay.com
- Freepik.com (с указанием автора)

Все изображения оптимизированы для веб-использования (800x600, качество 80%).
