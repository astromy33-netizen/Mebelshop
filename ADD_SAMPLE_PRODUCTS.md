# Добавление тестовых товаров с фотографиями

## Что было добавлено:

1. **15 тестовых товаров** с реальными фотографиями мебели из Unsplash
2. **Страница для добавления** тестовых товаров в админ-панели
3. **Автоматическое добавление** всех товаров в MockAPI

## Как использовать:

### Способ 1: Через админ-панель (рекомендуется)

1. Войдите в систему как администратор
2. Перейдите в `/admin/products`
3. Нажмите кнопку **"Add Samples"** (зеленая кнопка)
4. Или перейдите напрямую на `/admin/add-samples`
5. Нажмите **"Add Sample Products"**
6. Подождите, пока все товары будут добавлены
7. Вы увидите результаты добавления

### Способ 2: Через код

Если хотите добавить товары программно:

```javascript
import { addSampleProducts } from './utils/addSampleProducts';

// В компоненте или консоли браузера
const results = await addSampleProducts();
console.log(results);
```
